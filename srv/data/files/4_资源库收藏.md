## [Metalsmith](https://www.npmjs.com/package/metalsmith)
- 所有逻辑都由插件处理。您只需将它们链接在一起即可
```js
Metalsmith(__dirname)
  .use(markdown())
  .use(layouts('handlebars'))
  .build(function(err) {
    if (err) throw err;
    console.log('Build finished!');
  });
```
